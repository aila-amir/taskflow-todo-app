🗂 TaskFlow — To-Do List App

A beautiful, fully-featured to-do list application built with vanilla HTML, CSS, and JavaScript.

✨ Features
- Add, edit, and delete tasks
- Mark tasks as complete
- Filter by status (All / Active / Done)
- Filter by category (Work, Personal, Health, Learning)
- Priority levels (High, Medium, Low)
- Due dates with overdue warnings
- Real-time search
- Progress bar
- Data saved automatically (localStorage)
- Fully responsive on mobile

 🛠 Tech Stack
- HTML5
- CSS3 (animations, CSS variables, flexbox)
- Vanilla JavaScript (no frameworks)
- Browser localStorage for data persistence

📸 How to Run Locally
1. Clone this repository
2. Open `index.html` in your browser
3. That's it — no dependencies or build tools needed!
